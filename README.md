## 기술
로그인 / 회원가입
+ 유효성 검사
+ 빈값 예외 처리

메인 페이지
+ 지역 카테고리별 해당 정보 출력

숙소 등록 페이지
+ 관리자인 경우에만 숙소 등록 가능
+ 다중 이미지 파일 업로드
+ 해시 태그

상세 페이지
+ 좋아요 기능
+ 예약 시, 날짜와 인원 수에 맞는 총 가격 계산
+ 댓글 추가, 수정, 삭제

예약 리스트 페이지
+ 해당 유저의 예약 정보 리스트 출력
+ 예약 취소
